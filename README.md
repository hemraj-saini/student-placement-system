# 🎓 Student Placement Record System

This is a Python-based CLI project to manage student placement data.

## 🚀 Features
- Add student record  
- View all students  
- Search student by roll number  
- Update student details  
- Delete student record  
- View placement statistics  

## 🛠️ Tech Used
- Python  
- Pandas  

## 📊 Description
This project helps manage student data and analyze placement statistics such as placement rate.

## ▶️ How to Run

1. Install pandas:
   pip install pandas  

2. Open Jupyter Notebook:
   jupyter notebook  

3. Open the .ipynb file and run all cells
